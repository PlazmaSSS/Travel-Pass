# Travel-Pass

An Android application designed for modern travelers to manage boarding passes, track trip habits, and keep all essential travel itineraries in one reliable offline-first application.

## 🛠 Tech Stack & Architecture

- **Language:** Kotlin
- **Architecture:** MVVM (Model-View-ViewModel) pattern for clean and scalable code.
- **Local Database:** Room Persistence Library to securely store travel passes and logs offline.
- **UI:** Responsive XML layouts designed for quick access on the go.

## 🚀 Key Features

- **Itinerary Tracking:** Log and manage travel paths and user habits during trips.
- **Offline Reliability:** Seamless access to boarding data without an internet connection via Room.
- **Structured Architecture:** Decoupled UI and data layers ensuring fast and stable performance.
